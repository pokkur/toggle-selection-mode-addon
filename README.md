# Toggle Selection Mode Addon for Blender

A simple Blender addon, written by OpenAI, that allows you to toggle between vertex, edge, and face selection modes in Edit Mode with a single key press.

## Features

- Easily toggle between vertex, edge, and face selection modes
- Customizable hotkey
- Works with Blender 3.0 and later

## Installation

1. Download the latest version of the addon from the [Releases](https://github.com/pokkur/toggle-selection-mode-addon/releases) page.
2. Open Blender and go to Edit > Preferences > Add-ons.
3. Click the "Install" button and navigate to the downloaded `.zip` file, then click "Install Add-on".
4. Enable the addon by checking the box next to "Mesh: Toggle Selection Mode".

## Usage

1. Enter Edit Mode in Blender.
2. Press the configured hotkey (default is `Space`) to toggle between vertex, edge, and face selection modes.

## Customization

You can change the hotkey used to toggle selection modes by going to Edit > Preferences > Add-ons and expanding the settings of "Mesh: Toggle Selection Mode".

Note: If the chosen key is already assigned to another function, that function will be overridden and become unavailable.
